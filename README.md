MythCheck App  

Purpose of the App 

The purpose of this application is to test the user’s knowledge of common life hacks and determine whether they are true or myths. The app presents a series of questions and allows users to select answers, providing immediate feedback and a final score at the end. 

Design Considerations 

The app was designed with simplicity and usability in mind: 

Clean User Interface: 

A minimal layout was used to ensure the app is easy to understand and navigate.Consistent Layout: All screens follow a structured design with clear spacing and alignment. 

Readable Text: 

Font sizes and colors were chosen to ensure readability. 

User-Friendly Navigation: 

Buttons such as Start, Next, Review, and Back to Home make navigation simple. 

No Unnecessary Elements: 

Only required components were included to avoid confusion and keep the app efficient. 

 

Screens Overview 

Screen 1: Welcome Screen 

Displays the app title and description 

Introduces the user to the quiz 

Contains a Start Quiz button to begin 

 Screen 2: Question Screen 

Displays one question at a time 

Includes: 

Question number 

Progress bar 

Answer buttons (Hack / Myth) 

Feedback message 

Users select an answer and proceed using the Next button 

 

Screen 3: Score Screen 

Displays the user’s final score 

Shows a performance message 

Includes: 

Review Answers button 

Back to Home button 

 Functionality 

Questions are displayed one at a time 

User selects an answer (no typing required) 

Feedback is shown instantly 

Score is tracked throughout the quiz 

Final results are displayed at the end
